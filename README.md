# Orbital Chaos Detector - Galactic Dynamics (ocd-gd)

## Setup

### Perquisites

Before installing and setting up this repo (except with nix), make sure to install a **C++** compiler on your machine, along with **GNU** and **GSL**. All other optional libraries are installed on the go during setup.

**Caution:** In case of Nix, all Compilers and libraries are installed and compiled regardless of whether they are available on the machine locally or not.

### Commands

If you have Nix installed and setup, then simply run `nix develop`.
If you use uv, then run `uv sync`.
If you use poetry, then run `poetry install`.
If you use simply pip, then run `pip install`.
